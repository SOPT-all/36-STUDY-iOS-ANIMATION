## 📚 스터디 방식
- 모든 스터디원은 매주 실습 과제를 본인 브랜치에서 진행하고 각자 만든 `main/juri` 브랜치로 PR 날려주세요.
- 스터디 시간에 랜덤으로 해당 주차의 실습 과제를 발표합니다.
- PR에 어떤 방식으로 구현했는지 자세히 적어주면 좋아요! 


## 👤 브랜치 규칙
- 2개 만들어야 됩니다 !!!!!!
- `main-이름`에서 `week-이름` 브랜치를 만들어서 매주 실습 진행하면 됩니다 ! 

1) main/이름
- ex) `main-juri`

2) 주차/이름
- ex) `week1-juri`
